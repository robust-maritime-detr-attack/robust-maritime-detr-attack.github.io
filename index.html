Robust Physical Adversarial Patch Attacks Against Detection Transformers in Maritime Domains

Anonymous Contributors1,
1Anonymous

Abstract

Autonomous vehicles rely on modern computer vision (CV) techniques to execute tasks such as object detection, depth estimation, and path planning. Particularly, in maritime domains, CV models are required to perform surveillance, search and rescue, and disaster response missions. However, these models have shown to be susceptible to adversarial attacks, which can directly threaten operational safety, security, and decision-making by suppressing or falsifying detection in real-world scenarios. Motivated by these risks, we aim to study the effects of adversarial patch attacks against detection trans- formers (DeTR), using YOLOv8 as a baseline detector for com- parative simulation studies. This work proposes a reproducible framework for patch generation that maps a low-dimensional latent representation into a universal physical adversarial patch. Furthermore, we present DeTR-specific attack objectives to 1. suppress and encourage misclassification of a chosen target class, and 2, directly manipulate decoder cross-attention by maximizing attention mass on the patch region. This approach is then validated through simulated experiments trained on the MIT Sea Grant Marine Perception Dataset and custom datasets along with controlled physical tests on Autonomous Surface Vehicles (ASVs). We conclude that learned patches can significantly impact detection performance without the need to alter data at the pixel-level and that transformer-based detectors are especially vulnerable to patches that can target their attention mechanisms in both simulated and physical envi- ronments.

Patch Generation Architecture

Neural architecture used to generate universal adversarial patches from a low-dimensional latent vector. The decoder-based generator progressively upsamples the latent representation through convolutional blocks to produce a printable adversarial patch.

Sample Adversarial Patches

Examples of adversarial patches produced by the proposed generator. These patches are optimized to suppress or manipulate object detections while remaining robust to different placements and viewing conditions.

YOLO Simulation Test

Impact of the generated adversarial patches on a YOLOv8 detector in simulated maritime scenes. The results show that the patch can suppress boat detections or cause misclassification without directly covering the target object.

DeTR Simulation Test - Maritime Dataset

Simulation experiments using the Maritime Perception Dataset evaluate how the generated patches affect DeTR’s detection and attention maps. The results show significant reductions in detected boat counts and confidence scores when patches are applied.

DeTR Simulation Test - Custom Dataset

Additional simulations are performed using a custom dataset collected from the ASV platform. These experiments highlight how patches trained with different loss functions influence both detection outputs and attention behavior within the model.

Physical Experiments

We validate the simulated attack results through real-world tests using an Autonomous Surface Vehicle in a controlled lab environment. Printed adversarial patches are attached to the vessel to observe their impact on real-time detection.
